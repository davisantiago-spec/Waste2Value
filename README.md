# Waste2Value

📌 Descrição do Projeto

O Waste2Value é uma plataforma SaaS desenvolvida para ajudar empresas a gerenciar seus resíduos recicláveis de forma inteligente, transformando lixo em oportunidade de lucro.  O sistema permite registrar materiais descartados, calcular automaticamente o valor de mercado desses resíduos e fornecer relatórios que auxiliam na tomada de decisão.
Assim tendo um aplicativo para lucros de empresas, que tem itens para reciclar e ajudando assim o meio ambiente de alguma forma, pra não descartar lixos em lugares errados.

🎯 Objetivo

Promover a sustentabilidade e a economia circular, ajudando empresas a:
Reduzir desperdícios
Identificar oportunidades de lucro com reciclagem
Melhorar a gestão de resíduos

📌 Requisitos Funcionais (RF) 

- RF01 – Cadastro de Usuário
O sistema deve permitir que o usuário realize cadastro informando dados pessoais obrigatórios, como nome completo, e-mail, senha e telefone, garantindo validação dos dados inseridos.

- RF02 – Autenticação (Login)
O sistema deve permitir que o usuário realize login utilizando e-mail e senha previamente cadastrados, garantindo autenticação segura.

- RF03 – Cadastro de Material Reciclável
O sistema deve permitir que o usuário registre materiais recicláveis informando o tipo (plástico, papel, vidro, metal, etc.).

- RF04 – Registro de Quantidade
O sistema deve permitir que o usuário informe a quantidade do material reciclável em quilogramas (kg).

- RF05 – Cálculo do Valor Estimado
O sistema deve calcular automaticamente o valor estimado dos materiais com base no tipo e na quantidade informada, utilizando preços previamente definidos.

- RF06 – Exibição do Valor Calculado
O sistema deve apresentar ao usuário o valor estimado calculado de forma clara e imediata após o registro.

- RF07 – Visualização de Materiais Disponíveis
O sistema deve permitir que usuários do tipo comprador visualizem os materiais cadastrados disponíveis para negociação.

- RF08 – Envio de Propostas de Compra
O sistema deve permitir que compradores enviem propostas de compra para materiais cadastrados pelos usuários.

- RF09 – Gerenciamento de Propostas
O sistema deve permitir que o usuário visualize, aceite ou recuse propostas recebidas.

- RF10 – Comunicação entre Usuários
O sistema deve permitir a troca de mensagens entre usuários e compradores dentro da plataforma, garantindo comunicação direta para negociação.



⚙️ Requisitos Não Funcionais (RNF) 

- RNF01 – Desempenho

O sistema deve responder às requisições do usuário em até 2 segundos, mesmo com múltiplos acessos simultâneos.

 Justificativa: garantir boa experiência de uso.
 

- RNF02 – Disponibilidade

O sistema deve estar disponível 24 horas por dia, 7 dias por semana, com no mínimo 99% de uptime.

 Justificativa: empresas precisam acessar o sistema a qualquer momento.
 

- RNF03 – Segurança

O sistema deve garantir a proteção dos dados dos usuários por meio de criptografia de senhas e comunicação segura (HTTPS).

 Justificativa: proteger informações sensíveis e evitar vazamentos.
 

- RNF04 – Usabilidade

O sistema deve possuir interface intuitiva e de fácil navegação, permitindo uso por pessoas sem conhecimento técnico.

 Justificativa: facilitar adoção por diferentes tipos de usuários.
 

- RNF05 – Responsividade

O sistema deve ser compatível com dispositivos móveis e desktops, adaptando-se a diferentes tamanhos de tela.

 Justificativa: acesso em campo (empresas, coleta, etc.)

🌱 Contexto social e extensão universitária

O projeto está inserido no contexto de sustentabilidade e responsabilidade ambiental, contribuindo diretamente para a redução do descarte inadequado de resíduos.

Muitas empresas, principalmente de pequeno porte, não possuem controle sobre os materiais recicláveis que produzem, descartando-os sem aproveitamento econômico. Isso gera:

- Impacto ambiental negativo
- Desperdício de recursos reutilizáveis
- Perda de oportunidade de renda

Nesse sentido, o sistema propõe uma solução tecnológica acessível que:

- Incentiva a economia circular
- Promove a consciência ambiental
- Apoia a geração de renda por meio da reciclagem
